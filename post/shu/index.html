树

2020-07-21

是一种抽象数据类型（ADT）的数据结构，用来模拟具有树状结构性质的数据集合。它由n（n >= 1）个有限节点组成的一个具有层次关系的集合。

树具有以下特点：

  1. 每个节点有0个或者多个子节点
  2. 没有父节点的节点成为根节点
  3. 每个非根节点有且只有一个父节点
  4. 除了根节点以外，每个子结点可以分为多个不相交的子树。

树的术语：

有的术语我也没听说过，用到的不多。

树的种类：

二叉树通常以链表形式存储。

应用场景：

  1. xml， html等用到树
  2. 路由协议用到树
  3. mysql数据库用到树
  4. 文件系统的目录结构用到树
  5. AI算法都是树搜索，也包括decision tree
