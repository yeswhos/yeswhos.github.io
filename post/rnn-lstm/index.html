RNN, LSTM

2020-07-31

RNN：Recurrent Neural Network
LSTM：Long-Short Term Memory

RNN：
特点：对具有序列特性的数据有效，能挖掘数据中的时序信息以及语义信息。
（时序特性，符合时间顺序，逻辑顺序等，比如人说的一句话，是几个单词按照一定的逻辑顺序拼接起来的）

结构：

上图X （t - 1）代表一个单词的向量，其他的x也是同样的。W是每个时间点之间的权重矩阵。
这样，RNN可以记住每个时刻的信息，每个时刻的隐藏层不仅由该时刻的输入层决定，还由上一个时刻的隐藏层决定。
公式如下，Oｔ代表ｔ时刻的输出，Sｔ代表ｔ时刻的隐藏层。

总结:RNN每一个时刻的输出结果都与上一时的输入有很大关系.
如果把输入序列换个顺序,那得到的结果也将是截然不同的.可以处理序列数据,同时对序列也很敏感.
